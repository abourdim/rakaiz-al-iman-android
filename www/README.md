# Pillars of Faith — ركائز الإيمان بين العقل والقلب

Interactive trilingual web app based on Sheikh Mohammed al-Ghazali's book "Rakaiz al-Iman Bayn al-Aql wal-Qalb" (Pillars of Faith Between Mind and Heart).

## Features

- 20 pillar cards organized by theme (God, destiny, prophets, afterlife, science, balance)
- Each pillar: rational argument (mind) + spiritual insight (heart) + Quran verse + reflection
- "Build Your Faith" visual — each pillar lights up when completed
- Young Explorer (7-12) and Teen Scholar (13+) modes
- Quiz: "Who Wants to Be a Scholar?" — 15 questions with 3 lifelines
- XP and badge system
- Trilingual: Arabic, English, French
- 3 themes: Pillar (purple), Night, Light
- Audiobook narrator with karaoke
- PWA-ready

## Book Summary

The book explores the foundations of faith combining rational arguments and spiritual insights. Key themes include God's existence and absolute power, refuting atheism and the chance theory, divine destiny, prophethood, the Quran's miracles, resurrection, and the balance between material and spiritual life.
